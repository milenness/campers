# 🚐 TravelTrucks

## 🔗 [Live Demo](https://your-deployed-link.com)

**TravelTrucks** is a specialized web application for campervan rentals. The project provides a seamless user experience for discovering, filtering, and booking campers for road trip adventures, built with modern web technologies.

## 🚀 Key Features

* **Advanced Camper Catalog** — Browse a wide variety of campers with server-side pagination and dynamic filtering by location, body type, engine, and transmission.
* **Infinite Loading** — Seamlessly load more campers using **TanStack Query** (`useInfiniteQuery`) for an optimized browsing experience.
* **Detailed Camper Profiles** — Dedicated dynamic pages for each vehicle featuring image galleries (via **Swiper.js**), comprehensive technical specifications, and user reviews.
* **Interactive Booking System** — Integrated reservation form with validation and successful submission notifications.
* **Responsive & Semantic** — Built with **Next.js App Router** and **TypeScript**, ensuring a fast, SEO-friendly, and maintainable codebase.

## 🛠 Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **State & Data Fetching:** TanStack Query (React Query)
* **Styling:** CSS Modules
* **Key Libraries:** Swiper.js (for galleries), Axios (for API requests)
* **Deployment:** Vercel
